When you should use this component

You should use the text area component when you require users to enter more than a line of text.

When you should consider something else

Users can sometimes find it difficult giving open-ended answers. If your use case permits, instead of asking users a single complex question, it is advisable to break it up into a series of simplier questions. E.g using the checkbox component if you want users to give multiple answers from a predetermined set of options.
