Overview

Use our flexible grid system to structure website content. Our grid system is based on a twelve-column system, and is powered by flexbox

How it works

Our grid system uses a series of containers, rows, and columns to lay out and align content. Below is you will find examples on how all this comes together

12 columns

This is our baseline grid that we use, with size classes ranging from is-2 to is-12. In the following example we see the difference between is-2 and is-6.

Container

.sgds-container centers the container and gives it a maximum width of 1300px.

Row

.row sets a row within the container with appropriate spacing in place

Columns

.col is-[2-12] indicates the number of columns the item spans out of a possible 12 per row. If you want 3 equal-width columns across, use .col is-4 for each item (please see 3 columns section below). If you want a single column, use .col without using [is-2-12]
