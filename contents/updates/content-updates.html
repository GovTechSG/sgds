Migrating to v2

Last updated on July 14th, 2022

Yes, we are finally migrating to a new portal soon! The v2 will be released by the end of this quarter. Tune in here for further updates!

v2 Release Info:

  • v2 Portal with Bootstrap 5

    We will be incorporating Bootstrap 5 as our base framework for our v2 components. This will also be leveraged across upcoming projects.

  • React Storybook Library

    In addition to our v2 portal, we will also be releasing a React library version displayed in Storybook UI explorer to cater to more users. Upcoming projects such as Vue & Web Component will also be hosted in Storybook.

  • v1 Support to Continue

    Do note that v1 usage will still be supported. However, we encourage users to transition to v2 as we are actively building on new features within the platform and having more user feedback will help us serve our users better.
