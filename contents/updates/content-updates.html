Upgrading to v2

Last updated on July 21st, 2022

We will be upgrading to a new portal soon with a new base framework (details below) which will be released at the end of July 2022.

v2 Release Info:

  • SGDS v2 with Bootstrap 5+

    We will be leveraging Bootstrap v5+ as our base framework. This will be used for all components, patterns and templates.

  • SGDS ReactJS Storybook Library

    In addition, we will also be releasing a React version on Storybook. Upcoming support for other JS frameworks will be released in the upcoming months.

  • v1 Continued Support

    Take note that v1 of SGDS will still be supported. However, we do encourage users to migrate to v2 as we will be actively updating and improving our offerings.

    Feel free to contact us if you have any questions or would like to know more.
